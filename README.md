# usage

requires [git](https://git-scm.com/) and [node.js](https://nodejs.org/)

1. `git clone https://github.com/lilyshibe/mew.git`
2. `cd mew && npm i`
3. rename `config.js.example` to `config.js`, edit file to your liking
4. `npm start` to start server
